# 💳 Java Banking Application

## 📌 Overview

This project is a Java-based banking application developed using object-oriented programming (OOP) principles. It simulates real-world banking operations, allowing users to manage different types of accounts and perform basic financial transactions.

---

## ⚙️ Features

* Create different account types (Savings, Checking, Credit)
* Deposit and withdraw funds
* Balance tracking for each account
* Input validation to ensure data accuracy
* Basic transaction management system

---

## 🧠 Concepts Used

* Object-Oriented Programming (OOP)

  * Classes & Objects
  * Encapsulation
  * Inheritance (if used)
  * Method design
* Conditional logic and validation
* Basic data management

---

## 🏦 Account Types

* **Savings Account** – Supports deposits, withdrawals, and balance tracking
* **Checking Account** – Used for daily transactions
* **Credit Account** – Simulates borrowing with balance management logic
